Rock, Paper, Scissors


This is a game of rock, paper, scissors. You pick either rock, paper, or scissors and the computer will aswell. Rock beats scissors, scissors beats paper, and paper beats rock. Type in your choice in the box below.


Dice Game


This is a game of dice. Press the button to roll the dice. You win if the first roll is a natural (7, 11) and lose if it is craps (2, 3, 12). If a point is rolled (4, 5, 6, 8, 9, 10) it must be repeated before a 7 is thrown in order to win. If 7 is rolled before the point you lose.
